# License Plate Recognition Project

This repository contains a License Plate Recognition (LPR) system built using computer vision and machine learning techniques. The implementation is in the Jupyter Notebook:

**license_plate_rec.ipynb**

## Features
- Detect license plates from images
- Extract and preprocess plate regions
- OCR for character recognition
- Visual result outputs

## Technologies Used
- Python
- OpenCV
- Pytesseract / OCR
- Jupyter Notebook

## How to Run
1. Download this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt

3. Open the notebook:
   jupyter notebook license_plate_rec.ipynb
